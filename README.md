<h1 align="left">CÂMBIO FÁCIL - Aplicativo Android</h1>

###

<h4 align="left">O Câmbio Fácil é um aplicativo que permite ao usuário escolher uma moeda e digitar um valor para converter  em outra moeda utilizando uma base de dados da AwesomeAPI que atualiza o valor da taxa de câmbio comercial entre as moedas a cada 30 segundos.</h4>

###

<h2 align="left">Modo de Usar</h2>

###

<p align="left"><b>1)</b> Escolher, na primeira lista de opções, a moeda que deseja converter, ou seja, a moeda de origem;</p>

###

<p align="left"><b>2)</b> Digitar o valor que deseja converter;</p>

###

<p align="left"><b>3)</b> Escolher, na segunda lista de opções, a moeda de destino;</p>

###

<p align="left"><b>4)</b> Apertar o botão "Converter" para que o valor da conversão e outras informações, como a taxa de câmbio, apareçam na tela.</p>

###
